## 🚀 Getting Started

### Prerequisites

- Flutter SDK
- Dart SDK
- Android Studio
- Git

### Clone the Repository

```bash
git clone https://github.com/amalmk-codes/flutter_bootcamp.git
```

### Navigate to the Project

```bash
cd flutter_bootcamp/my_app
```

### Install Dependencies

```bash
flutter pub get
```

### Run the App

```bash
flutter run
```
